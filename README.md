## 🚀 Schnellstart

1. Repository klonen:
   ```bash
   git clone https://github.com/tuxpeople/toolbox.git
   cd toolbox
   ```

2. Gewünschtes Tool verwenden:
   ```bash
   cd crane_fqdn
   ./crane_fqdn.sh nginx:latest
   ```
